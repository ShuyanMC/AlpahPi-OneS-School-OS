# AlpahPi-OneS-School-OS
AlpahPi OneS Red Star School OS:使用mpy进行编程，制作的可用于核桃编程/量子兔的AlphaPi的OS，利用硬件特性，实现了Wi-Fi功能，将其变成了一个简易的MP3。适用于学生，内置小说阅读器，后续会利用Wi-Fi功能实现联网阅读。
